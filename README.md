# LLM-api-example

## Usage:  

### notebook


### backend
- Clone this repo
- pip install fastapi uvicorn transformers torch requests
- pip install fastapi[all] 
- See requirements.txt 
- Have your api key on .env
- Run server: uvicorn main:app --reload

### frontend
- cd to frontend folder
- npm install web-vitals
- npm install axios
- npm start
